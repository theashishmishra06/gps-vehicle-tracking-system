# GPS-Based Vehicle Tracking and Anti-Theft System

## Overview
An ESP32-based vehicle tracking and anti-theft system designed for real-time location monitoring and security alerts.

## Features
- Real-time GPS tracking
- Google Maps link generation
- SMS-based notifications using GSM
- Unauthorized movement detection
- Remote vehicle monitoring

## Hardware Components
- ESP32 Development Board
- NEO-6M GPS Module
- SIM800L GSM Module

## Software
- Arduino IDE
- Embedded C++
- TinyGPS++ Library
